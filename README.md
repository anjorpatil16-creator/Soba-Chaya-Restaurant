# 🍜 Soba Chaya — Unofficial Fan Website

An unofficial fan website for **Soba Chaya (そば茶屋)**, a handmade soba noodle restaurant in Tokushima Prefecture, Japan, owned by 97-year-old Ms. Mukai. Created as a study project.

---

## 📖 About

Soba Chaya (そば茶屋), established in 1968, is a beloved restaurant tucked in the mountains of Tokushima, Japan. The owner, Ms. Mukai, wakes up at 6am every day to handcraft fresh soba noodles. The shop is open **365 days a year**.

This website was built to practice frontend web development using HTML, CSS, and Bootstrap.

## ✨ Features

- **Image carousel** — Rotating banner showcasing the restaurant, noodles, and scenery
- **About section** — Story of Ms. Mukai and a quote from the owner
- **Photo gallery** — Cards displaying the menu, seating, and food
- **Menu modal** — Click "See Menu" to view the handwritten menu image in a lightbox
- **Footer** — Opening hours, address, attribution, and disclaimer

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/soba-chaya.git
   ```

2. **Navigate into the folder**
   ```bash
   cd soba-chaya
   ```

3. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to open it in your browser. No build tools or server required.
